# Configuración después de instalar

Para preparar Arch para el uso diario, se recomienda seguir estos pasos:

1. Crea un usuario nuevo, consulta: [crear usuario](./users/user.md)
2. Instala y configura el firewall, consulta: [instalar UFW](./UFW/)
3. Activa el repositorio multilib, consulta: [Repositorios oficiales](./repositories/repositories.md)
4. Para instalar y configurar las fuentes, consulta: [Fuentes](./fonts/fonts.md)

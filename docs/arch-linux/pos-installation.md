# Configuración después de instalar

Para preparar Arch para el uso diario, se recomienda seguir estos pasos:

1. Verifica que el microcódigo se haya cargado correctamente. Para más detalles, consulta: [Microcode](./microcode/microcode.md)
2. Instala los controladores de tu tarjeta gráfica según corresponda. Consulta: [NVIDIA](./display-drivers/nvidia.md), [AMD](./display-drivers/amd.md) o [Intel](./display-drivers/intel.md) según sea el caso.
3. Crea un usuario nuevo. Para más información, consulta: [crear usuario](./users/user.md)
4. Instala y configura el firewall. Consulta la guía en: [instalar UFW](./UFW/installation-UFW.md)
5. Activa el repositorio multilib. Consulta: [Repositorios oficiales](./repositories/repositories.md)
6. Para instalar yay y poder utilizar AUR, consulta: [yay](./aur/yay.md)
7. Para instalar y configurar las fuentes, revisa: [Fuentes](./fonts/fonts.md)
8. Instala Hyprland. Para ello, se recomienda consultar la guía: [Hyprland](../hyperland/getting-started/install-hyprland.md)

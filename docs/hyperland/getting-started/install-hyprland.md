# Instalar Hyprland

Para instalar Hyprland en Arch de la forma recomendada, realiza los siguientes pasos:

1. Ejecuta el siguiente comando `sudo pacman -S hyprland`

Ten presente que Hyprland no es un entorno de escritorio completo, sino un compositor y un conjunto de herramientas para que construyas tu propio entorno. Por ello, la instalación y configuración de aplicaciones y utilidades adicionales queda bajo tu responsabilidad.

Consulta el artículo [Tutorial base](master-tutorial.md) para dar tus primeros pasos.

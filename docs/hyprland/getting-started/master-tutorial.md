# Tutorial base

Una vez instalado Hyprland, sigue estos pasos esenciales:

1. Instala una terminal. Se recomienda `kitty`: [kitty](./terminal/kitty.md)
2. Lanza Hyprland desde TTY con `start-hyprland` (sin sudo)
3. Configura el agente de autenticaci√≥n: [hyprpolkitagent](../must-have/authentication-agent/hyprpolkitagent.md)
4. Instala XDG Desktop Portal: [xdg-desktop-portal-hyprland]()
5. Configura wallpapers: [hyprpaper](../must-have/wallpaper/)
6. Configura notificaciones: [mako](../must-have/notifications/mako.md)
7. Instala el lanzador de aplicaciones: [hyprlauncher](../must-have/application-launcher/hyprlauncher.md)
8. Configura el gestor de archivos: [yazi](../must-have/file-manager/yazi.md)
9. Instala Flatpack para aplicaciones: [Flatpack]()
10. Configura la barra de estado: [quickshell]()

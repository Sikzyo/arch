# Configuración para el editor Zed

## Extensiones

- **Catppuccin**: Tema de colores para el editor
- **Material Icon Theme**: Iconos para los archivos y carpetas

## Configuración del editor

Para ver las configuraciones completas, consulta el archivo [`settings.json`](./settings.json).

## Atajos de teclado (Keymaps)

Para ver la configuración de los atajos de teclado, consulta el archivo [`keymap.json`](./keymap.json).

---

## Instalación rápida

1. Instala las extensiones mencionadas desde el marketplace de Zed
2. Copia el contenido de `settings.json` en tu configuración de Zed
3. Copia el contenido de `keymap.json` en tu configuración de atajos
